# ExerciciosListas

## Lista de atividades 1

* Crie uma lista vazia. 
* Adicione 3 elementos à lista.
* Acesse o segundo elemento da lista.
* Percorra a lista e imprima cada elemento na tela.
* Remova o primeiro elemento da lista.
* Verifique se a lista contém um elemento específico.
* Ordene a lista em ordem crescente.
* Insira um novo elemento na segunda posição da lista.
* Limpe a lista.

## Lista de atividades 2

* Crie uma lista vazia.
* Adicione 5 elementos à lista.
* Crie uma nova lista contendo os elementos da primeira lista, mas em ordem inversa.
* Percorra a lista e imprima apenas os elementos pares.
* Remova todos os elementos ímpares da lista.
* Crie uma nova lista contendo apenas os elementos únicos da primeira lista.
* Ordene a lista em ordem decrescente.
* Insira um novo elemento na posição correta da lista, de forma a manter a lista ordenada.
* Limpe a lista.

## Lista de atividades 3

* Crie uma lista vazia.
* Adicione 10 elementos à lista.
* Crie uma nova lista contendo apenas os elementos da primeira lista que são divisíveis por 3.
* Percorra a lista e imprima apenas os elementos que são maiores que a média da lista.
* Remova todos os elementos repetidos da lista.
* Crie uma nova lista contendo apenas os elementos que aparecem um número ímpar de vezes na lista original.
* Ordene a lista em ordem decrescente, utilizando um algoritmo de ordenação customizado.
* Insira um novo elemento na posição correta da lista, utilizando uma busca binária.
* Limpe a lista.
